Student Management SPA (React + Node.js)

A Single Page Application (SPA) built using React for the frontend and Node.js + Express for the backend. Demonstrates API integration, CRUD operations, and responsive UI.

Project Overview

This project showcases a full-stack SPA where users can view and add student data dynamically. The frontend React app communicates with the backend Express API to fetch and manage student records in real-time.

Features

Display student list

Add new student via form

Dynamic updates without page reload (SPA)

Backend REST API using Node.js & Express

CORS enabled for frontend-backend communication

Technologies Used

Frontend: React, JSX, Fetch API

Backend: Node.js, Express.js

JSON for data transfer

CORS for cross-origin requests
